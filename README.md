# Tessera Snake

A real-time terminal Snake game built in C# with Tessera.

## Build

```powershell
dotnet build
```

## Run

```powershell
dotnet run
```

## Verify

```powershell
dotnet run -- --self-test
```

## Controls

- Arrow keys or WASD: move
- Space: restart after game over
- Ctrl+Q: quit
